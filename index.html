🧠 1. The Core Idempotent Engine

  • Smart State Parsing: Reads current running-config, compares to target settings, and only generates commands for exact missing or differing lines. Resilient against terminal emulator copy/paste spacing quirks.
  • Automated Rollback Generation: Builds a reverse script to perfectly undo changes and return ports to their original state.
  • Dynamic Interface Chunking: The Omni-Matrix engine evaluates every single port individually. It then smartly groups ports that share the exact same resulting commands into interface range chunks to keep scripts impeccably clean. Reduces chunk size automatically if Slow Paste is enabled.
  • Surgical Command Stripping: Automatically injects exact no commands to scrub old protocols before applying new ones.

🛡️ 2. Advanced Security & Protocol Management

  • Trunk Port Hard-Stop: Actively locks out ports detected as uplinks or trunks to prevent catastrophic network outages.
  • Omni-Matrix Assignment: Updates configurations dynamically. Set Global Master parameters to apply to all ports simultaneously, or use the Per-Port Override Grid (07) to surgically assign different VLANs and Security Protocols to specific interfaces within the same execution batch.
  • DOT1X Mode: Pushes 802.1X templates (dot1x pae authenticator & authentication port-control auto).
  • STICKY Mode: Pushes MAC Sticky templates (switchport port-security & ...mac-address sticky).
  • REMOVE Mode: Triggers Critical Danger Glitch. Explicitly strips existing configurations. Requires a mandatory compliance confirmation prompt. Rollback intelligently restores the deleted security.
  • Universal RESET: Automatically disabled unless port security is actually present. Wipes existing security clean. Security resets intentionally omit rollback commands, but if paired with a VLAN change, the engine will still independently roll back the VLAN.
  • Smart Port Bouncing: Calculates if a change requires shut / no shut to clear violation states, handling it in both scripts.

🚨 3. Real-Time Dynamic Warning Scanner

  • Critical Warning (Red): Detects Trunk/Uplink ports (missing switchport mode access) and executes a complete script lockout for those specific ports.
  • Dual-Security Conflict (Magenta): Detects ports infected with both 802.1X and Sticky MAC. Blocks "RESET" to force explicit protocol cleanup.
  • Fragmented Config (Yellow): Detects partial security templates (e.g., missing base commands) and prompts for repair.
  • Unsecured Port Notice (Cyan): Flags naked ports with zero port security configured.
  • Shutdown Notice (Orange): Lists currently shutdown ports to prevent troubleshooting confusion.

⚡ 4. Edge, State, & Output Controls

  • Independent EDGE Evaluation: Checks portfast and bpduguard enable separately. Only pushes missing lines.
  • Admin State Override: Includes UP (no shut), DWN (shut), BNC (explicit shut/no shut bounce), and LEAVE (preserve) toggles. Rollback intelligently reverses any forced state changes.
  • Cross-Panel Locking & Smart Idempotency: Modifying security automatically locks the Bounce button. UI buttons gray out if the engine detects the config already meets targeted conditions natively.
  • Omni-Drop Zone: Supports drag-and-drop ingestion of raw text highlights, IDE tabs, or `.txt` log files directly into the UI. Direct `[ DOWNLOAD ]` buttons export scripts named intelligently after the target hostname.
  • Chronos Planning Module: Real-time timezone converter with a built-in planning feature. Click the Local Time to pause the clock and input a custom execution window to immediately see it converted to your target timezone.
  • Haptic/Audio Engine: Triggers synthesized Web Audio confirmation on execution. Includes Dry Run Diff visualizer in the terminal output status string. Includes interactive dynamic network canvas background.

🎨 5. Zero-Trust Local Storage DB

  • Local Browser Cache: Safely stores Switch Hostnames and IP addresses strictly in your browser. Data never touches the internet.
  • Auto-Cross Reference: Automatically extracts hostnames from pasted terminal prompts and projects the target IP into the HUD.
  • Regex Smart Parser: Allows pasting raw data direct from Excel or IPAM tools regardless of column order.
  • Multi-Tenant HUD: Dynamically scans logs and generates individual tracking tags for every unique switch detected in the payload.
  • Auto-Merge Capability: Automatically appends new pastes to your existing master list without deleting your previous hosts.
