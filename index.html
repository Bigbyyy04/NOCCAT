🧠 1. The Core Idempotent Engine

  • Omni-Parse Technology: The engine automatically detects whether you have pasted standard configuration blocks or a raw list of interface names.
  • Smart Range Aggregation: To prevent console buffer overrun and reduce script size, the engine automatically detects consecutive targeted ports (e.g. Gi1/0/1, Gi1/0/2) and natively compresses them into perfectly formatted interface range commands.
  • Smart State Parsing (Config Mode): Reads current running-config, compares to target settings, and only generates commands for exact missing or differing lines.
  • Blind Force Processing (Raw Mode): If raw ports are detected, idempotency is dynamically disabled. The engine will forcefully apply your selected parameters without evaluating current states.
  • Automated Rollback Generation: Builds a reverse script to perfectly undo changes and return ports to their original state (or applies best-effort defaults for raw ports).
  • Surgical Command Stripping: Automatically injects exact no commands to scrub old protocols before applying new ones.

🛡️ 2. Advanced Security & Protocol Management

  • Trunk Port Hard-Stop: Actively locks out ports detected as uplinks or trunks to prevent catastrophic network outages (Automatically bypassed for Raw Ports).
  • Omni-Matrix Assignment: Updates configurations dynamically. Set Global Master parameters to apply to all ports simultaneously, or use the Per-Port Override Grid (07) to surgically assign different VLANs and Security Protocols to specific interfaces within the same execution batch.
  • DOT1X Mode: Pushes 802.1X templates.
  • STICKY Mode: Pushes MAC Sticky templates.
  • REMOVE Mode: Triggers Critical Danger Glitch. Explicitly strips existing configurations. Requires a mandatory compliance confirmation prompt.
  • Universal RESET: Wipes existing security clean. Security resets intentionally omit rollback commands.

🚨 3. Real-Time Dynamic Warning Scanner

  • Raw Port List Notice (Yellow): Alerts the engineer when Blind Override Mode is actively engaged. Includes a quick-toggle to force switchport mode access onto unknown blind targets.
  • Critical Warning (Red): Detects Trunk/Uplink ports (missing switchport mode access) and executes a complete script lockout for those specific ports.
  • Dual-Security Conflict (Magenta): Detects ports infected with both 802.1X and Sticky MAC. Blocks "RESET" to force explicit protocol cleanup.
  • Unsecured Port Notice (Cyan): Flags naked ports with zero port security configured.

⚡ 4. Edge, State, & Output Controls

  • Admin State Override: Includes UP, DWN, BNC, and LEAVE toggles. Rollback intelligently reverses any forced state changes.
  • Cross-Panel Locking & Smart Idempotency: Modifying security automatically locks the Bounce button. UI buttons gray out if the engine detects the config already meets targeted conditions natively.
  • Omni-Drop Zone: Supports drag-and-drop ingestion of raw text highlights, IDE tabs, or `.txt` log files directly into the UI. Direct `[ DOWNLOAD ]` buttons export scripts.
  • Omni-Chronos Persistent Module: A fully unrestricted, text-parseable dual-world clock that caches your preferred timezone and 12H/24H format to local storage.
