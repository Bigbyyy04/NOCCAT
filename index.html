⚙️ 1. Idempotent Execution Engine

  • State-Aware Compilation: Parses current `show run` configurations to evaluate existing states. Only outputs commands needed to reach the target state, completely ignoring parameters that already match.
  • Smart Range Aggregation: Automatically compresses sequential ports (e.g., Gi1/0/1, Gi1/0/2, Gi1/0/3) into optimized interface range Gi1/0/1-3 blocks to save terminal buffer space.
  • Automated Rollback Protocol: Dynamically calculates and generates the exact reverse script required to seamlessly undo applied changes and return interfaces to their pre-execution state.
  • Surgical Command Stripping: Intelligently prepends `no` commands (e.g., `no dot1x pae authenticator`) before applying new security protocols to prevent device rejection or ghost configurations.

📡 2. Omni-Parse Input & Drag-and-Drop

  • Standard Parsing: Accepts standard Cisco interface configuration blocks natively (e.g., interface Gi1/0/1...).
  • Blind Mode (Raw Ports): Auto-detects flat lists of ports without config data. Disables global settings to enforce safety, requires configuration via the explicit Advanced Matrix, and prevents arbitrary state bouncing.
  • Omni-Drop Zone: Seamlessly drag and drop `.txt` log files or raw text highlights directly into the input box for immediate parsing. Auto-clears cache on reload to prevent accidental execution of old data.

🛡️ 3. Security & Port Protection

  • Trunk Port Hard-Stop: Automatically detects uplinks/trunks (missing `switchport mode access`) and entirely locks them out of the script generation, preventing catastrophic network loops or outages.
  • DOT1X & STICKY Pushing: Instantly formats ports with 802.1X or Port Security MAC Sticky templates while removing conflicting protocols.
  • REMOVE Protocol: Systematically strips security to leave the port open. Triggers a mandatory Critical Danger validation modal to enforce IT CAB/Ticket compliance.
  • Universal RESET: Wipes existing security and immediately replaces it. During Blind Mode, dynamically limits resets to explicit protocol targets.
